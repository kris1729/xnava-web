# Xnava Enterprises Website

**xnava.in** — Corporate website for Xnava Enterprises, Lucknow.

## 🏢 About

Xnava Enterprises houses two educational units:

- **TechTrainx Labs** (`/techtrainx`) — 3-month placement bootcamp for engineering students
- **Chetna Academy** (`/chetna`) — Classes 5–12, JEE & NEET coaching

## 🛠 Tech Stack

| Layer    | Technology                          |
| -------- | ----------------------------------- |
| Frontend | React 18, Vite, Tailwind CSS, Framer Motion |
| Backend  | Node.js 20, Express 5, Mongoose 8   |
| Database | MongoDB Atlas                       |
| Email    | Resend API                          |
| Hosting  | Vercel                              |
